Three-Method Detection

Chronos runs three detection strategies in parallel and uses the highest-confidence result

Title Parsing

Fast • ~50% of detections

Regex extraction from window titles. Works well with apps that show full file paths (VS Code, Revit, AutoCAD).

File System Matching

Medium • ~30% of detections

Searches your project folders for the active filename. Resolves ambiguity when titles are truncated or generic.

Handle Enumeration

Precise • ~20% of detections • Patent Pending

Queries the OS for every file handle open by the active process. Gets the exact file path, even when the window title says nothing useful. 95–98% accuracy on Windows.

Frequently Asked Questions

What is handle enumeration?

Handle enumeration is Chronos's patent-pending technology that queries the operating system for every file handle open by the active process. Instead of guessing from the window title, it gets the exact file path — like C:\Cases\Baker-vs-County\Baker.pdf — achieving 95–98% detection accuracy on Windows.

How is this different from Toggl, Harvest, or Clockify?

Those tools rely on manual timers or basic window title parsing. Chronos is fully automatic — no start/stop buttons. It detects files at the OS level, classifies activity with local AI, and runs 100% offline. No data ever leaves your device.

Is my data safe?

Yes. Chronos stores only metadata (file paths, timestamps, project names) in an AES-256-GCM encrypted local database. File contents are never read. Nothing is transmitted. Designed for attorney-client privilege and HIPAA compliance.

Do I need an internet connection?

No. Chronos works completely offline. Internet is only needed for license activation and optional AI model downloads. All tracking, detection, and reporting runs locally.

Can I try Chronos before committing?

Yes — the free 14-day trial requires no credit card. Download directly from the pricing page and start tracking immediately. Your local database stays on your machine forever, even if you cancel.

Which industries is Chronos best for?

Any professional who bills by the hour: attorneys, expert witnesses, architects, BIM managers, consultants, freelancers, and software developers. If you track billable time, Chronos can automate it.
